# 公演结算 — 后端实现详细说明

## 1. 评级计算公式

### 选手个人得分

```
normalizedVocalW = song.vocalWeight / (vocalWeight + danceWeight + charmWeight)
normalizedDanceW = song.danceWeight / total
normalizedCharmW = song.charmWeight / total

attributeScore = player.vocal × normalizedVocalW
              + player.dance × normalizedDanceW
              + player.charm × normalizedCharmW
              // 范围: 0~100

difficultyFactor = 1 - (song.difficulty - 1) × 0.1
                  // difficulty=1 → 1.0, difficulty=5 → 0.6

performanceBonus = player.performanceValue × 2
                  // performanceValue 范围 -5~15, 加分范围 -10~30

playerFinalScore = Math.round(attributeScore × difficultyFactor + performanceBonus)
                  // 截断到 0~120 范围
```

### 选手评级映射

| 评级 | 名称 | 阈值 |
|------|------|------|
| S | 完美舞台 | score ≥ 85 |
| A | 出色表现 | score ≥ 65 |
| B | 稳定发挥 | score ≥ 45 |
| C | 略有不足 | score ≥ 25 |
| D | 失误较多 | score < 25 |

### 团队得分

```
teamAvgScore = Math.round(average of all members' playerFinalScore)
teamBonus = random(0, 20)      // 团队配合随机加成
teamScore = teamAvgScore + teamBonus
```

团队评级使用与选手相同的阈值映射。

### 最终票数（用于排名）

```
finalVotes = 500 + teamScore × 3 + random(-10, 20)
```

## 2. POST /api/performance/calculate

**用途**：后端根据本轮数据计算公演结果，并持久化存储。

**请求**：
```
POST /api/performance/calculate
Content-Type: application/json
Authorization: Bearer <token>

{
  "roundId": "round-1"
}
```

**后端处理逻辑**（伪代码）：

```
function handleCalculate(roundId):
  // 1. 根据 roundId 获取本轮数据
  teams = db.getTeamsByRound(roundId)           // RoundTeam[]
  teamSongs = db.getTeamSongsByRound(roundId)   // TeamSong[]
  players = db.getPlayersByRound(roundId)       // User[] (不含管理员)

  teamResults = []
  playerResults = []

  // 2. 对每个队伍计算
  for each team in teams:
    if team.members is empty: continue

    teamSong = teamSongs.find(s => s.teamId === team.id)
    if teamSong is null: continue

    song = teamSong.song
    songWeights = normalizeWeights(song.vocalWeight, song.danceWeight, song.charmWeight)
    difficultyFactor = 1 - (song.difficulty - 1) * 0.1

    memberScores = []

    // 3. 对每个成员计算
    for each member in team.members:
      player = players.find(p => p.id === member.playerId)
      if player is null: continue

      // 选手个人得分
      attrScore = player.attributes.vocal * songWeights.vocal
                + player.attributes.dance * songWeights.dance
                + player.attributes.charm * songWeights.charm

      perfValue = db.getPlayerPerformanceValue(roundId, member.playerId)
                  // 从 performance_player 表获取，范围 -5~15
                  // 如果未生成则使用 random(-5, 15)

      playerScore = Math.round(attrScore * difficultyFactor + perfValue * 2)
      playerScore = Math.max(0, Math.min(120, playerScore))

      rating = getRating(playerScore)   // S/A/B/C/D
      ratingText = getRatingText(rating)

      memberScores.push({ playerId, playerName, playerScore, rating, ratingText, perfValue })

    // 4. 团队得分
    avgScore = Math.round(sum(memberScores.score) / memberScores.length)
    teamBonus = random(0, 20)
    teamScore = avgScore + teamBonus
    teamRating = getRating(teamScore)

    // 5. 最终票数
    rankFluctuation = random(-10, 20)
    finalVotes = 500 + teamScore * 3 + rankFluctuation

    teamResults.push({
      roundId, teamId: team.id, teamName: team.name,
      songId: teamSong.songId, songName: song.name,
      memberCount: team.members.length,
      baseVotes: 500,
      attributeVotes: teamScore * 3,
      performanceVotes: rankFluctuation,
      eventVotes: teamBonus,
      finalVotes: Math.max(0, finalVotes),
      rank: 0,
      status: 'calculated',
      teamScore, teamRating, teamRatingText,
      songWeights,
      teamAttributes: { vocal, dance, charm },
      playerPerformances: [...]
    })

    // 6. 选手结果
    rankInTeam = 1
    for each ms in memberScores:
      playerResults.push({
        roundId, playerId, playerName, teamId, teamName,
        performanceValue: ms.perfValue,
        playerScore: ms.score,
        stageRating: ms.rating, stageRatingText: ms.ratingText,
        contribution: ms.score,
        rankInTeam: rankInTeam++
      })

  // 7. 排序
  teamResults.sort(by finalVotes descending)
  teamResults.forEach((team, index) => team.rank = index + 1)

  // 8. 关联选手到队伍结果
  for each team in teamResults:
    team.playerPerformances = playerResults.filter(p => p.teamId === team.teamId)

  // 9. 持久化存储
  db.saveTeamResults(roundId, teamResults)
  db.savePlayerResults(roundId, playerResults)

  return { success: true, teams: teamResults, players: playerResults }


// 辅助函数
function normalizeWeights(v, d, c):
  total = v + d + c
  if total == 0: return { vocal: 0.34, dance: 0.33, charm: 0.33 }
  return { vocal: v/total, dance: d/total, charm: c/total }

function getRating(score):
  if score >= 85: return 'S'
  if score >= 65: return 'A'
  if score >= 45: return 'B'
  if score >= 25: return 'C'
  return 'D'

function getRatingText(rating):
  return { S:'完美舞台', A:'出色表现', B:'稳定发挥', C:'略有不足', D:'失误较多' }[rating]
```

**响应**：
```json
{
  "success": true,
  "data": {
    "teamResults": [ /* TeamPerformanceResult[] */ ],
    "playerResults": [ /* PlayerPerformanceResult[] */ ]
  }
}
```

## 4. GET /api/performance/result?roundId=round-1

**用途**：获取指定轮次已结算的公演结果（用于页面刷新后恢复）。

**请求**：
```
GET /api/performance/result?roundId=round-1
```

**响应**：
```json
{
  "success": true,
  "data": {
    "teamResults": [],
    "playerResults": []
  }
}
```

如果该轮次尚未结算，返回空数组。

## 5. 新增数据库表

### performance_player — 选手发挥值

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| round_id | string | 轮次ID |
| player_id | string | 选手ID |
| performance_value | integer | 发挥值 (-5~15) |
| created_at | timestamp | 创建时间 |

SQL:
```sql
CREATE TABLE performance_player (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  round_id VARCHAR(50) NOT NULL,
  player_id VARCHAR(50) NOT NULL,
  performance_value INTEGER NOT NULL CHECK (performance_value >= -5 AND performance_value <= 15),
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(round_id, player_id)
);
```

### performance_result_team — 队伍结算结果

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| round_id | string | 轮次ID |
| team_id | string | 队伍ID |
| data | JSONB | TeamPerformanceResult 完整 JSON |
| created_at | timestamp | 创建时间 |

### performance_result_player — 选手结算结果

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| round_id | string | 轮次ID |
| player_id | string | 选手ID |
| data | JSONB | PlayerPerformanceResult 完整 JSON |
| created_at | timestamp | 创建时间 |
