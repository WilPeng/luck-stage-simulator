# 💕 恋综「喜爱度」投送与结算规则说明

## 一、游戏整体流程

每轮游戏按以下 **3 个阶段** 循环进行：

| 顺序 | 阶段 | 说明 |
|:---:|------|------|
| 1 | **喜爱值投送** | 选手给其他选手分配喜爱值 |
| 2 | **配对结算** | 根据投送结果进行配对 |
| 3 | **淘汰** | 落单者被淘汰 |

一轮结束后自动进入下一轮的「喜爱值投送」，共进行 **若干轮**（默认）。

---

## 二、喜爱值投送规则

### 2.1 预算（Budget）

- 每轮每位活跃选手会获得一个 **随机预算**，范围：**100 ~ 160 点**
- 预算由系统在进入「喜爱值投送」阶段时自动生成
- 每个人每轮的预算**不同**，随机独立

### 2.2 投送方式

选手需要将本轮的所有预算 **分配给自己以外的其他选手**，规则如下：

| 规则 | 说明 |
|------|------|
| **必须用完预算** | 分配给所有人的点数之和 **必须等于** 系统分配的预算，不能多也不能少 |
| **不能给自己** | 自己的喜爱值不能投给自己 |
| **值必须互不相同** | 给每个选手的数值 **不能重复**，例如不能同时给 A 和 B 都投 30 点 |
| **每人至少 1 点** | 每个活跃选手都至少分到 1 点 |
| **仅限活跃选手** | 只能投给当前状态为 `active` 的其他选手（已被淘汰的选手不可投） |

> **举例**：如果你的预算是 120，场上有 A、B、C 三位其他选手，你可以分配如：
> - A → 10，B → 50，C → 60（总和 = 120，值均不同 ✅）
> - 不能：A → 40，B → 40，C → 40（值重复 ❌）
> - 不能：A → 30，B → 30（总和 ≠ 120 ❌）

### 2.3 提交方式

- **选手自行提交**：在选手端「喜爱值投送」页面填写数值后提交
- **管理员可撤回**：管理员可以撤回某位选手的投送记录，撤回后可重新投送

### 2.4 提交时机

- **只能在「喜爱值投送」阶段提交**，其他阶段（配对结算、淘汰）不可提交

---

## 三、配对结算规则

当管理员推进到「配对结算」阶段时，系统会自动根据本轮所有选手的投送数据进行计算。

### 3.1 计算步骤

#### 第一步：统计每位选手收到的喜爱值总数

将本轮所有投送记录中，每个选手作为「被投送人」获得的点数相加。

#### 第二步：选出「单身汉」

**收获喜爱值最少** 的那位选手成为本轮 **单身汉**。单身汉本轮将进入观察室，对其余CP的节目录制有一定程度的影响。

> 注意：只有 **已投票的活跃选手** 参与配对结算。没提交投送的选手不参与。

#### 第三步：剩余选手贪心配对

1. 计算剩余所有选手 **两两之间** 的「互相投送值之和」（A 给 B 的数值 + B 给 A 的数值）
2. 按互相投送值 **从高到低排序**
3. 从最高值开始依次配对，**已经配对的选手跳过**
4. 如果剩余人数为奇数，最后一位自动归入下一轮处理

#### 第四步：保存结果

- 配对成功：形成若干对 CP
- 未配对：产生一位单身汉

### 3.2 配对公式总结

```
互相投送值 = A 给 B 的喜爱值 + B 给 A 的喜爱值

配对顺序：按互相投送值从高到低，贪心匹配（取最高的先配对）

单身汉 = 收获喜爱值总数最少的选手
```

---

## 四、淘汰规则

- 配对阶段产生的 **CP** 有一组会在「淘汰」阶段被淘汰
- 被淘汰后状态变为 `eliminated`，不再参与后续轮次
- 淘汰后剩余选手进入下一轮，系统重新生成新的预算

---

## 五、常见问题

**Q：预算没用完可以提交吗？**
A：不可以，投送总和必须 **严格等于** 系统分配的预算。

**Q：可以投给自己吗？**
A：不可以，不能给自己投送喜爱值。

**Q：给两个人的数值可以相同吗？**
A：不可以，每个选手获得的数值必须 **互不相同**。
